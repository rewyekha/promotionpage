MAS 500 Build Schedule
MAS 500 builds take place Monday and Thursday of each week. The Monday build is designated a "non-schema change" build, meaning that items promoted for this build should not have any dependency on a change to the database schema. The Thursday build, or "schema change build", will include database schema changes in addition to promoted items.
