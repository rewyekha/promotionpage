Introduction

  • The Builder Web Pages are used to maintain the Builder database used in the weekly MAS 500 Client and Database build process.
    •  
  • The Project Maintenance page is used to add, maintain, or remove entries for the builder database. These entries represent Visual SourceSafe files or projects, and include such items as Visual Studio projects, Database Objects, Help files, Crystal report files, and other documents. A project entry in the builder database can consist of an entire SourceSafe project tree, or an individual file.
  • The Promote Project page is used to indicate that an item has a new version updated in SourceSafe which is to be extracted in the next client or database build.
